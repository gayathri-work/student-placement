# student-placement
Exploratory Data Analysis of Student Placement Dataset using Python, Pandas, NumPy, Matplotlib, and Seaborn.


## Project Overview

This project performs **Exploratory Data Analysis (EDA)** on a Student Placement dataset collected from **Kaggle**. The objective is to analyze various academic and skill-based factors that influence student placement outcomes. By leveraging Python's data analysis and visualization libraries, the project uncovers patterns and insights that can help educational institutions and students understand the key factors contributing to successful placements.

---

##  Objectives

- Analyze the student placement dataset.
- Perform data cleaning and preprocessing.
- Explore the distribution of different student attributes.
- Identify factors affecting placement status.
- Visualize relationships between variables.
- Generate meaningful insights through exploratory data analysis.

---

## Dataset

- **Source:** Kaggle
- **Type:** Student Placement Dataset
- **Format:** CSV

The dataset contains information related to students' academic performance, technical skills, internships, communication skills, projects, and placement status.

## Key Insights

- Academic performance plays a significant role in placement outcomes.
- Internship experience positively impacts placement opportunities.
- Technical and communication skills contribute to employability.
- Students with better overall profiles have a higher likelihood of getting placed.
- Visualizations help identify trends and relationships among different variables.
